QR Code Generator For Social Media Platforms:

This project is a web-based application developed using python,flask as a frame work and XAMPP server.It enables users to generate QR codes for social media platforms like Facebook,Instagram,YouTube and plain text.Users must login to create QR codes,Which are automatically saved in a designated folder for easy retrieval.Each generated QR code includes a Platform-specific logo in the middle,such as the Facebook,Instagram or youtube logo,making it visually distinctive and brand-specific.The application uses MYSQL for data management and features a responsive interface ,providing an effective tool for social media promotion and branding.

Guess My Number website:

This project is a fun and interactive number-guessing game developed using HTML,CSS,JavaScript. In this game,the user tries to guess a randomly generated number within a specified range, with the system providing feedback such as "too high" or "too low" after each guess.The project includes features like attempt tracking, a reset option, and customizable difficulty levels, making it an engaging way to demonstrate the use of basic web technologies for creating interactive applications.
